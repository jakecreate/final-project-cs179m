This is a web application. We did not deploy the web app due to the financial cost. However, if this were a real client, we would have deployed it.

To run this web app, follow these steps:
1) Clone the repository
2) Navigate to the project repository
3) pip install flask (install flask 3.1.2)
4) pip install numpy (install numpy version 2.2.4)
5) make sure to have Python version 3.12.2 or later installed
6) run 'python ./src/endpoints.py' in the terminal
7) click on the link it generates. this will open the website in a browser.
